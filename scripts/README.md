# Scripts d'automatisation

## Scripts PowerShell recommandés (Windows)
- **START_WITH_VENV.ps1** - Script principal de démarrage avec environnement virtuel
- **SETUP_VENV_WINDOWS.ps1** - Configuration automatique de l'environnement de développement

## Scripts Batch (compatibilité)
- **START_SIMPLE.bat** - Démarrage simple sans configuration avancée
- **start_validation_test.bat** - Script de tests de validation

### Scripts de développement
- **start_backend.bat** - Démarrage backend uniquement
- **start_frontend.bat** - Démarrage frontend uniquement

## Utilisation recommandée
1. Exécuter `SETUP_VENV_WINDOWS.ps1` pour la première installation
2. Utiliser `START_WITH_VENV.ps1` pour les démarrages quotidiens
