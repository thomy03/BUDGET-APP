# Clean Project

Clean and organize project files for better structure.

## Instructions

Analyze the project root directory and perform the following cleanup tasks:
1. Update CLAUDE.md in order to add the lastest features updates and not continue using old script.
2. Identify unneeded files and delete them
3. Merge duplicate files when multiple files contain the same content
4. Summarize or move unclassified files to appropriate dedicated folders
5. Organize the project structure for better maintainability

Focus on files at the root that are not properly classified or organized.
