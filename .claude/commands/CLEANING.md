# Clean Project

Clean and organize project files for better structure.

## Instructions

Analyze the project root directory and perform the following cleanup tasks:
1. Identify unneeded files and delete them
2. Merge duplicate files when multiple files contain the same content
3. Summarize or move unclassified files to appropriate dedicated folders
4. Organize the project structure for better maintainability

Focus on files at the root that are not properly classified or organized.
